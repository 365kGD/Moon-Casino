# Moon Casino

Static fake-money arcade site for GitHub Pages.

## Run

Open `index.html` directly in a browser or publish the repository with GitHub Pages.

## Notes

- Demo coins only.
- No real money, no cashout, no backend.
- The lobby opens first and links to the casino or the simulated crypto trading area.
- Trading includes simulated price graphs and per-coin stop-loss rules for holding value or price.
- Moon Predictor includes a licensed trade mode for simulated dip-buying, stop loss, and take profit.
- Admin panel opens from the browser console with `openAdminPanel()` or `admin()`. Password: `123456`.
- Press `H` to leave the game and open Google.
- GitHub Pages entrypoint: `index.html`.
